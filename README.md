Hey, So you have a chromebook right?
And you have stumbled upon termux right?

Well this is for you than.
Introducing XFCT - Chromebook Edition.

This was built for Chromebooks in mind due to the X86_64. But I do believe this works for all devices (ARM, X86, Etc...)

What this allows you to do it get a simple interface on termux and install xfc4, ubuntu, pulseaudio, and required apps to run Termux-X11.

Speaking of which!
You do require these three apps.

Termux-X11 - [https://github.com/termux/termux-x11](url) or 

Termux (Obviously.) - [https://f-droid.org/en/packages/com.termux/](url) or https://github.com/termux/termux-app

Termux API - [https://f-droid.org/en/packages/com.termux.api/](url) or [https://github.com/termux/termux-api](url)

___________________________________________________________________________________________________________________

Steps to use installation

1. Download Prerequiset Apps

2. Download XFCT_BASE

3. Paste this into your Termux Folder

4. Enter Termux

5. Run ls (To confirm the file is in the folder

6. Run chmod +x XFCE_BASE

7. Run ./XFCE_BASE
